IP Bridge & Web Portal

The Alertex IP Bridge allows you to monitor and control your Alertex system from one central location.

  • Monitor and control the Alertex system from one central location
  • Up to 1km line of sight wireless transmission range
  • Initiate site-wide or individual unit tests
  • IP66 weather rated

NXIBRIDGE | IP Bridge

The Alertex IP Bridge connects the Alertex system to a local network, allowing the system to be monitored and controlled from one computer via your web browser.

The bridge can be powered by 5 volts DC or via PoE and can be quickly configured over Bluetooth using your smartphone. This allows users to obtain the IP address, set the site and subnet code, and connect the IP bridge to the local Wi-Fi network.

The Alertex IP Bridge has a wireless transmission range of up to 1km line of sight, which can be extended using an AE434 aerial to assist when dealing with difficult sites. The Alertex IP Bridge facilitates a system of 64 total units and is IP66 weather rated, making it suitable for outdoor use.

  • Send email and text notifications to key staff
  • Activate or reset system from central location
  • Configure voice annunciators
  • Initiate silent tests

The Alertex Web App

The Alertex IP Bridge creates its own portal that can be accessed via your web browser. The Alertex portal allows users to monitor and control their Alertex system from one central location. The portal is password protected.

End users can see which units have been activated, and can reset the system via the Alertex portal post-activation. Silent tests can be initiated via the portal, and voice annunciator audio parameters can be configured.

Event notifications can be sent via email or text message to key staff (using Twilio, account required), and can be configured to send different alerts to specific staff members, such as activations, missed calls, or low-battery alerts.
